工程文件位于Calculator文件夹当中，Setup.exe为部署程式，采用InstallShield LE部署，安装程式相对较大（50MB）。Calculator.sln为解决方案文件，包括了部署项目的文件内容。

运行平台：Windows
功能开发工具：Visual Studio 2013 Ultimate Update3
GUI开发工具：Blend for Visual Studio 2013
部署解决方案：InstallShield LE
生成目标平台：32位
目标框架：.Net Framework 4.5 Full Package, IronRuby 1.1
输出类型：Windows 应用程序
项目类型：WPF工程
窗体风格：WPF默认风格（系统）
按钮风格：Dynamic cRecourse
前端语言：XAML
后台语言：C#

文件信息：
	Ioperate.cs             计算操作接口，用于判断读取对象是否为数字、比较运算符优先级以及扫描表达式
	Expression.cs           计算操作类，实现Ioperate接口，实现计算当中对栈调用的各种方法
	MainWindow.xaml         XAML定义的GUI文件，用于实现计算器的GUI窗体
	MainWindow.xaml.cs      后台控制文件，用于传递和控制各种数据
	icon.ico		图标文件

使用方法：
	基本功能：左键单击按钮可进行交互
	科学功能：左键单击按钮可进行交互
	存储功能：S1、S2、S3等以及X、Y、Z六个按钮支持此功能，当显示框中存在合法数据时，右键单击按钮即可将当前运算结果记录到单击的变量中。同时，使用左键单击按钮可读取其存储的数据
	函数功能：Fun、SFun和EFun三个按钮来操作函数功能，计算模式下左键点击SFun按钮可进入函数模式，此时左击X、Y、Z将不再读取其内容而是将其作为变量放入表达式，如：输入X、+、2显示框的到字符串“X+2”；输入完表达式后点击EFun可切换回计算模式并将表达式存储起来；在计算模式下左键点击Fun按钮可输出函数表达式，并提取X，Y，Z的值进行函数计算


功能特色：
	1.支持大部分科学计算，包括常用三角函数、次幂、二次方根、自然对数、倒数以及小数据下的阶乘运算
	2.支持π和e两个常用数据计算
	3.GUI界面交互功能
	4.支持负数转换，程序中允许+-6的情况存在
	5.支持对输出结果的复制
	6.支持对输出结果的继续运算，再次输入不会清空结果
	7.支持浮点数运算
	10.自动匹配括号，不允许输入空内容括号，同时不允许输入超过左括号数目的右括号
	11.对于无效0的情况，该程序会自动去除，如0000002和2.00000都会自动格式成2
	12.支持对中间数据进行存储，最多可存储六个数据，之后可随时调用
	13.使用布尔类型单链表存储左括号状态，如果为true则为进行匹配，反之已匹配，快速查找括号状态
	14.支持两种模式，计算模式和函数模式，即可计算自定义函数表达式，最多可包含X、Y、Z三个变量
	15.使用Ruby嵌入脚本进行阶乘运算，可以直接进行大于2000的数的阶乘运算（根据堆栈空间确定上限），且运算快速
	16.支持通过Ruby脚本进行的进制转换功能

缺点：
	1.对高精度数据的运算支持很差
	2.不支持在表达式中插入二次方根运算
	3.不支持中括号[]和大括号{}的使用
	4.不支持科学计数法e10的表示方法，该程序当中的e指无理数e=2.718281828
	5.不支持10为底和2为底的常用对数运算
	6.不支持在表达式中插入自然对数
	7.不支持对其他进制的计算，不包括针对二进制的与或非等运算
	8.代码抽象不够彻底，设计较差，耦合度太高
